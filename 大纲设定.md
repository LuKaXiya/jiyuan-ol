# 《纪元OL》小说设定与大纲

---

## 一、核心设定

### 游戏背景
- **游戏名**：《纪元OL》
- **类型**：全息沉浸式虚拟网游（VR）
- **特色**：职业系统、副本、公会战、国战

### 装备等级体系
```
白板（新手）→ 黑铁器 → 绿岩器 → 青铜器 → 白银器
→ 黄金器 → 紫霖器 → 灵禹器 → 天翎器
→ 圣昀器 → 仙霖器 → 鬼灵器 → 次神器 → 国器
→ 神器★ → 神器★★ → 神器★★★ → 神器★★★★ → 神器★★★★★
→ 史诗神器 → 天下国器
```

### 职业转职体系

#### 普通职业路线（10级一转）
| 等级 | 剑士路线 | 法师路线 | 弓手路线 | 刺客路线 |
|------|---------|---------|---------|---------|
| 1级 | 新手 | 新手 | 新手 | 新手 |
| 10级 | 初级剑士 | 初级法师 | 初级弓手 | 初级刺客 |
| 30级 | 剑客 | 魔导士 | 狙击手 | 暗影 |
| 50级 | 剑灵 | 大魔导师 | 箭神 | 舞者 |
| 80级 | 剑圣 | 法圣 | 箭圣 | 影圣 |
| 100级 | 剑神 | 法神 | 箭神 | 影神 |

#### 隐藏职业：虚空剑宗
- **觉醒条件**：发现隐藏地图「虚空裂隙」
- **特殊**：无需一转，觉醒即特殊职业
- **技能解锁方式**：打怪顿悟（不靠等级）

## 四、大纲（分四卷，共约900章）

### 第一卷 · 虚空觉醒（1-100章）

**现实线**：大学毕业，入职小公司，被边缘化

**游戏线**：
- 新手村憋屈，12级不一转
- 偶获「虚空剑宗」隐藏职业
- 第一次小团战胜利，初步崭露头角

**感情线**：未开启

**结束点**：获得第一件紫霖器，声望小涨

---

### 第二卷 · 小有名气（101-300章）

**现实线**：被御姐会长（北冥）注意到，职场开始有转机

**游戏线**：
- 加入苍穹公会边缘成员，崭露头角
- 第一次大规模副本战
- 顿悟解锁「虚空斩」

**感情线**：柠檬茶在现实中认识主角，开始黏人

**结束点**：公会内部权力斗争，第一次夺城战

---

### 第三卷 · 风云人物（301-600章）

**现实线**：身份被质疑，职场危机

**游戏线**：
- 天翎城权力争斗，被迫选边站
- 顿悟解锁「影无痕」
- 被背叛，跌入谷底
- 虚空种族完全觉醒

**感情线**：三角关系升温，北冥身世揭露

**结束点**：主角跌入谷底，但获得更强力量

---

### 第四卷 · 巅峰与真相（601-900章）

**现实线**：主角真实身份揭晓——不是普通人

**游戏线**：
- 国战开启，对抗最强敌国
- 虚拟与现实边界崩塌
- 终极觉醒「一剑隔世」

**感情线**：最终选择

**结局**：开放式/或有深意的收尾

---

## 五、升级与战斗节奏

### 核心原则
- 每20-30章完成一次大阶段升级
- 每个大阶段有明确目标：解锁技能 / 获得装备 / 声望提升
- 永远不让主角在舒适区待太久

### 每章字数
| 章节类型 | 字数 | 占比 |
|---------|------|------|
| 日常/过渡章 | 2000-2500字 | 约60% |
| 情节推进章 | 2500-3000字 | 约30% |
| 关键大战章 | 4000-6000字 | 约10% |

## 六、去AI味检查清单

每章写完自检：
```
□ 删掉所有"完美的"形容词
□ 短句比例 ≥ 40%
□ 对话推进剧情了吗？还是在凑字数？
□ 主角有没有说出只有他才会说的话（性格标签）？
□ 上章结尾的钩子有没有接上？
□ 等级/装备/声望有没有更新？
□ 感情线进度有没有变化？
□ 本章埋了哪个伏笔？
```
